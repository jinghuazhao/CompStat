# ChatGPT under Flask

As ususal, this is furnished with [`app.py`](app.py) and [`config.py`](config.py) with [`templates`](templates) in its simplest form.

```
Flask/
├── app.py
├── config.py
├── README.md (this file)
└── templates
    └── index.html
```

By default, `python app.py` will enable `http://127.0.0.1:5000`.
